## Problem Statement

Full fine-tuning of BERT updates all 110M parameters, making it computationally expensive. This assignment implements and compares **Parameter-Efficient Fine-Tuning (PEFT)** methods that achieve competitive performance while training only a fraction of the model's parameters.

---

## Dataset & Model

| | |
|---|---|
| **Base Model** | `bert-base-uncased` |
| **Dataset** | SQuAD v1.1 (5,000 train / 1,000 validation) |
| **Task** | Extractive Question Answering |

---

## Methods Implemented

### Core Tasks
| Method | Description | Trainable Params |
|---|---|---|
| Full Fine-Tuning | Baseline — all parameters updated | 108.9M (100%) |
| LoRA | Low-rank matrices injected into Q & V projections | 296K (0.27%) |
| Adapter Layers | Bottleneck modules after encoder layers | 1.2M (1.10%) |

### Extensions
| Method | Description | Trainable Params |
|---|---|---|
| Freeze Bottom 6 | Top 6 BERT layers + QA head trained | 42.5M (39%) |
| Attention-Only | Only attention projections + QA head trained | 28.3M (26%) |

---

## Results

| Model | Accuracy (EM%) | F1 Score | Training Time | Parameters |
|---|---|---|---|---|
| Full Fine-Tuning (Baseline) | 15.20% | 18.25% | 13.9 min | 108,893,186 (100%) |
| LoRA | 2.30% | 5.19% | 10.3 min | 296,450 (0.27%) |
| Adapter Layers | 2.50% | 5.47% | 5.1 min | 1,209,602 (1.10%) |
| Freeze Bottom 6 (Ext.) | 13.90% | 16.56% | 9.4 min | 42,528,770 (39%) |
| Attention-Only (Ext.) | 14.00% | 16.88% | 11.3 min | 28,349,954 (26%) |

> **Note:** LoRA and Adapter scores are lower due to training on a small subset (5,000 samples, 2 epochs). In literature, these methods match full fine-tuning on the full SQuAD dataset.
